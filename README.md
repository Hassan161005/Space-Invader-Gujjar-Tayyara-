# Space-Invader-Gujjar-Tayyara-
This is a console version of Space Invaders game written in C++ and utilizing the help.h library to handle the screen. The game involves the use of nested loops, arrays and in file handling to come up with an interesting gameplay.
These characteristics are some of the major ones:

Dynamic Grid of Aliens and: A grid of aliens travels across the screen. When one of the walls is approached, they turn and move in the opposite direction, going one row nearer to the player. The player is the winner, as they defeated all aliens and they could not reach the bottom.

Shooting System & Lives: The player in-control of a shooter is able to fire bullets towards the top. Each time some alien collides the player, he or she loses one life.

Score Tracking: The overall number of aliens hit will increase.

The pause and save function: The user may stop the game at any time and save his progress on a file.

Top 5 High Scores: The highest scores in the game are recorded using file handling with the first five winners in it.
